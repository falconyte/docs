## Welcome

The Falconyte Public API gives your integrations access to contacts, event
ingestion, and webhook management features. The reference pages in this section
are grouped by domain:

- Contacts API – manage email contacts, perform bulk actions, and retrieve unsubscribe activity.
- Events API – ingest custom lifecycle events and review automated campaign decisions.
- Webhooks API – configure outbound webhooks, inspect delivery history, and simulate payloads.
- Ping – simple health check endpoint.

## Authentication

All endpoints require an API key. Include the key issued to your workspace with
every request:

```http
X-API-Key: <your-api-key>
```

Requests without a valid key receive a `401 Unauthorized` response.
