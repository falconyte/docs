# Falconyte Docs

This repository powers the Falconyte documentation site at [https://docs.falconyte.dev/](https://docs.falconyte.dev/).

## Repository structure

The repo includes:

- Guides for getting started and customizing Falconyte
- Reference material for our public API
- Reusable snippets, images, and other shared assets

## Contributing

1. Create a feature branch for your documentation updates.
2. Make edits in the `docs/` directory.
3. Open a pull request for review.

If you have questions or need a hand, email us at [support@falconyte.com](mailto:support@falconyte.com).
